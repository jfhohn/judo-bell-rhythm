# Roadmap

## Planned Enhancements

### Priority
- [ ] Export/import schedules (JSON) - Backup and share schedule configurations
- [ ] Improved bell timing accuracy - Ensure bells always fire precisely at section end and 2-minute mark

### Nice to Have
- [ ] Custom sound uploads - Allow instructors to use their own audio files for bells

## Completed
See [changelog.md](./changelog.md) for shipped features.

## Out of Scope
The following features have been considered but are not planned:
- PWA support
- Multi-device sync / cloud storage
- Custom color picker for sections
- Voice announcements
- Student-facing display mode
- Analytics/usage tracking
- Keyboard shortcuts
